Test de vitesse de Network Monitor

Le test de vitesse de Network Monitor est un test simple qui n'est pas aussi précis que des tests effectués par d'autres applis qui se spécialisent en tests de vitesse.

Network Monitor est une application complètement gratuite, sans publicité, maintenue par une amatrice, et l'auteur de cette appli ne peut payer pour un serveur dédié, pour soutenir des tests de vitesse de milliers d'utilisateurs.

Par conséquent, dans Network Monitor, vous devez choisir un serveur sur lequel vous effectuez le test de vitesse.

Test de réception (http)

Pour le test de réception, vous choisissez l'URL d'un fichier à télécharger. Plus gros le fichier est, plus précis le test sera, mais plus la possibilité sera de dépasser votre quota de données, si vous en avez un.

Test d'émission (ftp)

Pour le test d'émission, Network Monitor va envoyer le fichier téléchargé pour le test de réception, sur un serveur FTP. Vous devez avoir accès à un serveur FTP et renseigner les identifiants, si vous souhaitez exécuter le test d'émission.

Si vous le souhaitez, vous pouvez exécuter le test de réception sans exécuter le test d'émission.

Vous en voulez plus ?

Si vous préférez une appli spécialisée en tests de vitesse, n'hésitez pas à chercher sur le Play Store.
