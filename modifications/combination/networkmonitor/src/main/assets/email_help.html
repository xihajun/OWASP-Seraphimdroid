E-mail help

I'm not receiving my e-mails

If you have configured Network Monitor to send yourself reports by e-mail using a Gmail account, you may need to configure Gmail to grant Network Monitor SMTP access.

There are two ways to allow Network Monitor to send mail using your Gmail account.

Recommended: 2-step verification

Enable 2-step verification, and then generate an app password which you can enter in Network Monitor instead of your Gmail password. You can do this here:
  • Enable 2-step verification
  • Generate an app password

Less secure:

If you haven't enabled 2-step verification in Gmail, you can allow all "less secure" apps to access Gmail. In this case, you enter your Gmail password inside Network Monitor.


Please note that enabling access to less secure applications can make your Gmail account more vulnerable.
