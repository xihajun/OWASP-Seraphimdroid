Prueba básica de velocidad de Network Monitor

La prueba de velocidad de Network Monitor es una prueba simple que no es tan precisa como pruebas realizadas por otras aplicaciones que se especializan en pruebas de velocidad.

Network Monitor es una aplicación completamente gratis, sin publicidad, mantenida por una aficionada, y la autora de esta aplicación no puede pagar por un servidor dedicado, para soportar pruebas de velocidad de miles de usuarios.

Por tanto, en Network Monitor, tienes que escoger un servidor sobre el cual realizarás la prueba de velocidad.

Prueba de bajada (http)

Para la prueba de bajada, escoges el URL de un fichero a descargar. Más grande el fichero es, más precisa la prueba será, pero más la posibilidad será de exceder tu cuota de datos, si tienes una.

Prueba de subida (ftp)

Para la prueba de subida, Network Monitor subirá el fichero descargado por la prueba de bajada, en un servidor FTP. Necesitas acceso a un servidor FTP y tienes que rellenar los credenciales de ingreso, si deseas ejecutar la prueba de subida.

Si deseas, puedes realizar solamente la prueba de bajada sin ejecutar la prueba de subida.
