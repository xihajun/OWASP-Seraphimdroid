Aide avec l'envoi des courriels

Je ne reçois pas de courriel

Si vous avez configuré Network Monitor pour qu'il vous envoie des rapports par courriel via votre compte Gmail, vous devez configurer Gmail pour donner un accès SMTP à Network Monitor.

Vous avez deux possibilités pour autoriser Network Monitor à envoyer du courriel avec votre compte Gmail.

Recommandé : validation en deux étapes

Activer la validation en deux étapes, puis générer un mot de passe d'application que vous pouvez saisir dans Network Monitor au lieu de votre mot de passe Gmail. Vous pouvez faire ceci ici :
  • Activer la validation en deux étapes
  • Générer un mot de passe d'application

Moins sécurisé :

Si vous n'avez pas activé la validation en deux étapes, vous pouvez autoriser un accès à Gmail pour toutes les applications "moins sécurisées". Dans ce cas, vous saisissez votre mot de passe Gmail dans Network Monitor.


Veuillez noter que donner accès aux applications moins sécurisées rend votre compte Gmail plus vulnérable.
